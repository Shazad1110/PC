# 📱 پڕۆژەی لاپەڕەی ئەپڵیکەیشن - شابڵۆد

## 📋 پێرستی فایلەکان

```
project/
│
├── index.html       # فایلی HTML سەرەکی
├── style.css        # فایلی ستایلی CSS
├── script.js        # فایلی JavaScript
├── app-data.json    # فایلی ڕێکخستن بۆ زانیاریی ئەپ
├── server.py        # سێرڤەری Python
└── README.md        # ئەم فایلە
```

## 🚀 چۆنیەتیی بەکارهێنان

### ڕێگای یەکەم: کردنەوەی راستەوخۆ
1. فایلی `index.html` بکەرەوە لە وێبگەڕەکەت
2. هەموو شتێک کاردەکات!

### ڕێگای دووەم: بەکارهێنانی Python Server
```bash
python server.py
```
دواتر بڕۆ بۆ: http://localhost:8000

## ✏️ چۆنیەتیی گۆڕینی زانیاریەکان

### گۆڕینی ناو، وێنە و وەسف:
فایلی `app-data.json` بکەرەوە و ئەم زانیارانە بگۆڕە:

```json
{
  "appName": "ناوی ئەپەکەت لێرە",
  "appIcon": "لینکی وێنەی ئایکۆن",
  "appDescription": "وەسفی ئەپەکەت لێرە بنووسە",
  "downloadLink": "لینکی App Store",
  "tutorialLink": "لینکی فێرکاری یان یوتیوب"
}
```

## 🎨 گۆڕینی ڕەنگەکان

فایلی `style.css` بکەرەوە و ئەم بەشانە بگۆڕە:

### ڕەنگی پاشبنەما:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### ڕەنگی دوگمەی دابەزاندن:
```css
.download-btn {
    background: #007AFF;  /* ڕەنگەکە لێرە بگۆڕە */
}
```

### ڕەنگی دوگمەی فێرکاری:
```css
.tutorial-btn {
    background: #2d3748;  /* ڕەنگەکە لێرە بگۆڕە */
}
```

## 📸 گۆڕینی وێنەی ئایکۆن

دوو ڕێگا هەیە:

**ڕێگای یەکەم (لینک):**
لە `app-data.json`:
```json
"appIcon": "https://example.com/your-icon.png"
```

**ڕێگای دووەم (فایلی ناوخۆیی):**
1. وێنەکەت دابنێ لە هەمان بوخچەی پڕۆژەکە
2. لە `app-data.json`:
```json
"appIcon": "my-app-icon.png"
```

## 🔗 گۆڕینی لینکەکان

### لینکی دابەزاندن:
```json
"downloadLink": "https://apps.apple.com/app/id123456789"
```

### لینکی فێرکاری:
```json
"tutorialLink": "https://youtube.com/watch?v=xxxxx"
```
